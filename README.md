# 👾 Gremlin

> Your users will break it anyway. Gremlin finds it first.

Gremlin is an open-source autonomous QA agent that spawns AI-powered user personas and unleashes them on your product. Each persona navigates independently, makes real decisions, hits dead ends, and finds bugs - without you writing a single test script.
