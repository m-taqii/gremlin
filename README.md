# 👾 Qlaw

> Claw through bugs before your users do.

Qlaw is an open-source autonomous QA agent that spawns AI-powered user agents and unleashes them on your product. Each agent navigates independently, makes real decisions, hits dead ends, and finds bugs - without you writing a single test script.
